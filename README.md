# Scrap Blend Optimiser Dashboard

Real-time dashboard displaying output from `optimiser_v3.py`.

## Setup

1. **Copy JSON file**: Place `dashboard_data.json` (generated by `optimiser_v3.py`) into this directory
2. **Install dependencies** (optional): `npm install`
3. **Local testing**: `npx http-server` then open `http://localhost:8080`

## How It Works

- `optimiser_v3.py` generates `dashboard_data.json`
- This dashboard fetches and displays the JSON data in a interactive webpage
- Deploy to Netlify for public access

## Deployment to Netlify

### Option 1: Via GitHub (Recommended)

```bash
# Initialize git repo
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/scrap-blend-optimiser-dashboard.git
git branch -M main
git push -u origin main
```

Then:
1. Go to [netlify.com](https://netlify.com)
2. Click "New site from Git"
3. Select your GitHub repository
4. Build settings:
   - Build command: (leave blank)
   - Publish directory: `.`
5. Click "Deploy"

### Option 2: Direct Deploy

```bash
npm install -g netlify-cli
netlify deploy --prod
```

## Auto-Update JSON

Schedule `optimiser_v3.py` to run daily:

### Windows Task Scheduler
1. Create a `.bat` file:
```batch
cd C:\Users\achaube\Downloads
python optimiser_v3.py
copy dashboard_data.json "C:\Users\achaube\Downloads\netlify-dashboard\"
git -C "C:\Users\achaube\Downloads\netlify-dashboard" add dashboard_data.json
git -C "C:\Users\achaube\Downloads\netlify-dashboard" commit -m "Update dashboard data"
git -C "C:\Users\achaube\Downloads\netlify-dashboard" push
```

2. Schedule it with Task Scheduler
